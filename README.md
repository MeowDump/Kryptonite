# Android Display Tweaker
A lightweight Magisk/KernelSU/APatch module that adjusts screen saturation and contrast in real-time using SurfaceFlinger calls. Includes a WebUI with live preview for easy tuning.

## Features

- **Real-time adjustments** : Changes apply instantly without reboot
- **WebUI with live preview** : Tune settings visually via browser
- **Persistent config** : Saves your preferences across reboots
- **Reset capability** : One-click restore to defaults
